Focus states

Keyboard and assistive-tech users see a visible focus ring. We use :focus-visible (with :focus:not(:focus-visible) to remove outline on mouse click). Outline: 2px solid var(--accent-indigo), 2px offset. Buttons and pill nav links use white outline on accent background for contrast. Applied to: .link-accent, .site-footer__text a, .site-footer__nav a, .nav__link, .btn.

Layout

Container

Main content width and horizontal margins

:root { --container-max: 1280px; --container-margin: 72px; } main, .site-header__inner, .site-footer__inner { max-width: var(--container-max); margin-left: auto; margin-right: auto; } main { margin: 0 var(--container-margin) 80px; } .site-header__inner, .site-footer__inner { margin: 0 var(--container-margin); }

Reading width (measure)

Comfortable line length for body text: aim for 45–75 characters per line. .hero .body-large uses max-width: 42rem; .case-hero__description uses max-width: 52rem. Apply similar constraints to long-form content.

See Typography / body-large and case hero description in styles.css.

Sections

Section wrappers and modifiers for index page layout

.section base margin. .section__header wraps title + subtitle (margin-bottom). .section--work extra top margin. .section--recommendations top margin. .section--anchor 1px height for anchor targets.

Spacing

Section gap

Vertical gap between major case study sections. Used by .case-block and .case-block--objective (margin-top).

:root { --space-section: 120px; --space-block: 80px; --space-stack: 24px; } .case-block, .case-block--objective { margin-top: var(--space-section); }

Spacing variables

Token purpose and usage

--space-section — gap between case study sections (120px). --space-block — legacy block spacing (80px). --space-stack — in-block vertical stack (24px). Adjust --space-section in :root to change section breathing room site-wide.
