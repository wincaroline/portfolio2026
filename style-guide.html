Spacing

Section gap

Vertical gap between major case study sections. Used by .case-block and .case-block--objective (margin-top).

:root { --space-section: 80px; --space-block: 56px; --space-stack: 24px; } .case-block, .case-block--objective { margin-top: var(--space-section); }

Spacing variables

Token purpose and usage

--space-section — gap between case study sections (80px). --space-block — legacy block spacing (56px). --space-stack — in-block vertical stack (24px). Adjust --space-section in :root to change section breathing room site-wide.
