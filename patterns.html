Case study shell

Full page structure for a new case study. Copy from components/case-study-shell.html. For long-form case studies use disney-case-study.html as reference: hero + My Role, Jump To nav, case-block sections, and footer with Say hello + nav.

Structure

body.case-body → header-case → main.case-main → footer. Paste hero (optionally with case-hero__role), case-jump-nav, then case-block sections. Footer: site-footer__copy (Say hello), site-footer__brand, site-footer__nav (Work, About, Resume, Play).

<body class="case-body">
  <!-- PASTE: components/header-case.html -->
  <main class="case-main">
    <!-- PASTE: case-hero (optional case-hero__role), case-jump-nav, case-block sections -->
  </main>
  <!-- PASTE: footer with site-footer__copy, site-footer__brand, site-footer__nav -->
</body>

Two-column content (case-block--two-columns)

Two equal columns; each has optional image above, then h3 and paragraph(s). Use for side-by-side comparison or dual topics.

Column 1 heading

First column body text. Image above is optional—omit the figure to show only heading and copy.

Column 2 heading

Second column body text. Each column can have an optional image above the h3.
